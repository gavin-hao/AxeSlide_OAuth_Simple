获取文件列表

api/v1/Author/Articles?skip={skip}&rows={rows}

参数

参数名 参数类型 说明
skip int 用于分页获取，跳过的行数
rows int 用于分页获取，要获取的记录行数

获取文件下载地址

api/v1/Article/{id}/downloadurl

参数

参数名 参数类型 说明
id long 作品id

返回值

						{
  "Value": "http://xxxxxx.com/xxx.dbk,文件下载链接",
  "Succeed": true,
  "Message": "sample string 3",
  "ErrCode": 0
}
